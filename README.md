# Toushik Banik — Personal Portfolio

Personal portfolio website built with vanilla HTML, CSS Grid, and a custom Swiss design system.

**Live:** https://toushikbanik123.github.io

## Pages

| Route | Description |
|-------|-------------|
| `index.html` | Home / Hero |
| `about.html` | About me |
| `work.html` | Projects |
| `systems.html` | Tech stack |
| `learning.html` | Education |
| `experience.html` | Work history |
| `case-medjeex.html` | Medjeex case study |
| `case-haatak.html` | Haatak case study |

## Stack

- HTML5 + CSS Grid (no framework)
- Custom Swiss design system (`assets/swiss.css`)
- Vanilla JS (`assets/swiss.js`)
- Hosted on GitHub Pages
