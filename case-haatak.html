04.01 / Problem

Live app, / tired UI, / conversion / leaking.

Haatak was already live with a working SafeGold integration and real users buying gold. But the UI was carrying weight from an earlier iteration and the team wanted a full redesign — auth, buy-gold, daily savings, withdraw, refer-and-earn, spin-the-wheel rewards. The SafeGold contract was solid; only the surface needed rebuilding.

04.02 / Approach

Engineering / Trust at / Scale.

Haatak is a high-security digital gold investment platform backed by SafeGold APIs and Razorpay. Users transact and vault 24K digital gold. I was contracted to architect the complete mobile presentation layer, building out the live-ticker dashboards, strict onboarding funnels, and the critical checkout flows.

In fintech, engineering translates directly into trust. Dropped frames or sluggish inputs destroy user confidence. The live gold price ticker had to poll and paint smoothly without blocking the main thread. The Razorpay integration demanded absolute state consistency—handling edge cases like OS interrupts, network drops mid-transaction, and webhook delays without ever putting the UI into an ambiguous state.
